## 수정 방법

각 파일에 다음과 같은 패턴으로 수정:

```dart
// 1. Import 추가
import 'package:lionsns/l10n/app_localizations.dart';

// 2. build 메서드 시작 부분에 추가
final l10n = AppLocalizations.of(context)!;

// 3. 하드코딩된 문자열 교체
Text('로그인') → Text(l10n.login)
'게시글 작성' → l10n.postCreate
```

## 참고사항
- 일부 파일은 동적으로 생성되는 에러 메시지가 있어 완전한 다국어 적용이 어려울 수 있음
- 데이터 소스 파일의 debugPrint 메시지는 제외해도 됨 (디버그용)
- 모든 사용자에게 보이는 UI 문자열만 다국어 적용하면 됨
